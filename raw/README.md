# raw/

Put source material here. `raw/papers/` is for PDFs and their extracted markdown text, `raw/web/` is for web-clipped sources, `raw/notes/` is for your rough notes, and `raw/images/` is for figures or screenshots. The LLM may read this directory, but should not rewrite your curated source material except when explicitly extracting a PDF to a companion `.md` file or updating the generated catalog files.
